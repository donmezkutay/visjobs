# visjobs

Visjobs offers plotting effective variables in effective way using atmospheric models.

## 1.Installation

#### 1.1. Dependencies
- xarray
- siphon
- pandas
- numpy
- matplotlib
- cartopy
- datetime

#### 1.2. Pypi Installation
- pip install visjobs

#### 1.3. Github Clone
- git clone https://github.com/donmezkutay/visjobs
